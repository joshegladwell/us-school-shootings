About this project

This website features a data science course project for the Data Science and Analytics Master's course at Georgetown University. The intent of the project is to examine the problem of school shootings in the United States using a data-driven approach.

Every step of this project has been carefully recorded and documented so that anybody with the resources could theoretically replicate it. All of the code used for the project is saved on this GitHub repository, which is currently private but will be publicized at the conclusion of the project. All of the data used for the project can be found on the repository as well, with the exception of datasets that do not permit redistribution according to the terms of use. These datasets are identified in the Data Gathering section of the website.

The website was designed entirely from scratch using HTML/CSS, with no use of templates.
